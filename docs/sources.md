# Seznam hlavních zdrojů dat a referencí

Tento dokument obsahuje jednotný přehled všech zdrojů, ze kterých Asistent IK OVS čerpá data, pravidla a metodické informace.
Každý zdroj má svůj alias, který se používá napříč požadavky (`REQ_XX.md`), mapováním i exporty.
